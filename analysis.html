Why This Matters for Your Decision

The misconceptions in your current document bundle Process Automation with Workzone as if they're inseparable. This creates a false dichotomy that limits your options. In reality, you have THREE distinct routes, not two, because these components are independent.

Process Automation Decision

Independent of UI layer choice

  • Use PA if you need standard workflow tooling
  • Use PA if you want built-in logging
  • Use PA if workflows are complex with approvals
  • Skip PA only if workflows are very simple

Workzone Decision

Independent of workflow layer choice

  • Use Workzone if you need portal aggregation
  • Use Workzone if you need federated content
  • Use Workzone if building enterprise-wide portal
  • Skip Workzone if simple app launcher is sufficient
